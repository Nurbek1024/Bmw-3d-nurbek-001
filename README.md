# PRO 3D BMW Racing

An exciting 3D racing game built with Three.js!

## Features

- 🏎️ 3D car racing gameplay
- 📱 Mobile touch controls (Left, Right, Boost)
- ⌨️ Keyboard support (Arrow keys + Space)
- 💾 High score tracking with localStorage
- 🎮 Progressive difficulty (speed increases over time)
- 🚀 Boost mechanic for faster movement

## How to Play

1. Open `index.html` in a web browser
2. Use **Arrow Keys** (or **mobile buttons**) to move left/right
3. Press **Space** (or **Boost button**) to temporarily increase speed
4. Avoid red enemy cars
5. Survive as long as possible to increase your score!

## Controls

### Keyboard
- **← / →** - Move left/right
- **Space** - Boost speed

### Mobile
- **⬅ Button** - Move left
- **➡ Button** - Move right
- **🚀 Button** - Boost speed

## Technologies

- **Three.js** - 3D graphics library
- **HTML5** - Structure
- **CSS3** - Styling
- **JavaScript (Vanilla)** - Game logic
